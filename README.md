##Requirements:
````
JDK 8
Maven 3.6.2
Node 12.14.0
````

##How to install
````
cd <app_folder>
mvn clean verify
````

##How to run

Backend
````
cd <app_folder>
java -jar target/tester-matching-1.0.0.jar
````
Frontend
````
cd <app_folder>/src/main/javascript/dist
ng serve
````

Open browser and enter URL: http://localhost:4200
